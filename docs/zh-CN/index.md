# 介绍

> 「最好的涩图插件！」
>    —— Shigma

Booru 是一个基于 [Koishi](https://koishi.chat/) 的图片检索插件，支持多个图源站点。

:::tip
你需要安装至少一个图源插件才能使用 Booru 插件。
:::

## 特性

- 支持多个[图源站点](#图源插件)
- 支持多种发送方式
- 支持随机检索

## 图源插件

Booru 插件支持多个图源站点，每个图源站点都有一个对应的插件。目前支持的图源站点有：
