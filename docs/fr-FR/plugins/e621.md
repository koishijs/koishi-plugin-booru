### 搜索设置

<br>

#### userAgent

- 类型：`string`
- 默认值：`Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/114.0.0.0 Safari/537.36 Edg/114.0.1823.37`

请求时所使用的 User-Agent

#### endpoint

- 类型：`string`
- 默认值：`https://e621.net`

e621/e926 的 API 地址。
